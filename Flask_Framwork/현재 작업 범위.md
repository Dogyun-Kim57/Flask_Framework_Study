
### 현재까지 작업한 트리 구조
```
minimalapp/
├─ run.py
│  # 프로젝트 실행 진입점
│  # Flask 앱을 생성하고 SocketIO 서버를 실행한다.
│  # host, port, debug 설정을 통해 로컬/외부 접속 테스트의 기준이 되는 파일이다.
│
├─ yolov8n.pt
│  # YOLOv8 경량 모델 가중치 파일
│  # 실시간 카메라 프레임에서 객체 탐지를 수행할 때 사용된다.
│
├─ app/
│  ├─ __init__.py
│  │  # Flask 애플리케이션 팩토리
│  │  # 앱 생성, Config 적용, DB/Migrate/Mail/SocketIO 초기화,
│  │  # 전체 Blueprint 등록과 실시간 Socket 이벤트 연결을 담당한다.
│  │
│  ├─ config.py
│  │  # 프로젝트 전체 환경설정 파일
│  │  # DB, 메일, 업로드, RTSP 카메라, 국가교통정보센터 CCTV API 설정을 관리한다.
│  │
│  ├─ extensions.py
│  │  # Flask 확장 객체 관리 파일
│  │  # db, migrate, mail, socketio 객체를 생성해 앱 초기화 시 연결한다.
│  │
│  ├─ onvif_test.py
│  │  # CCTV/ONVIF/RTSP 연결 테스트용 파일
│  │  # 실제 서비스보다는 카메라 주소 확인과 디버깅 용도에 가깝다.
│  │
│  ├─ common/
│  │  ├─ auth.py
│  │  │  # 로그인 여부와 관리자 권한을 확인하는 공통 인증 모듈
│  │  │  # login_required, admin_required, 현재 사용자 정보 조회 함수를 제공한다.
│  │  │
│  │  ├─ exceptions.py
│  │  │  # 커스텀 예외 정의 파일
│  │  │  # 서비스 계층에서 검증 실패 시 ValidationError 등을 발생시킨다.
│  │  │
│  │  ├─ response.py
│  │  │  # 공통 응답 처리 모듈
│  │  │  # HTML 렌더링, 리다이렉트, JSON 성공/실패 응답 형식을 통일한다.
│  │  │
│  │  ├─ session_manager.py
│  │  │  # 세션 관리 전담 모듈
│  │  │  # 로그인 시 user_id, username, email, role을 저장하고 로그아웃 시 세션을 정리한다.
│  │  │
│  │  └─ utils.py
│  │     # 공통 유틸리티 함수 모음
│  │     # 입력값 정리 등 여러 서비스에서 반복 사용하는 보조 기능을 담당한다.
│  │
│  ├─ models/
│  │  ├─ user.py
│  │  │  # 회원 테이블 모델
│  │  │  # 아이디, 이메일, 비밀번호 해시, 권한, 가입일을 저장한다.
│  │  │
│  │  ├─ contact.py
│  │  │  # 문의 테이블 모델
│  │  │  # 문의자 정보, 문의 내용, 관리자 답변, 작성 시간을 저장한다.
│  │  │
│  │  ├─ post.py
│  │  │  # 게시글 테이블 모델
│  │  │  # 제목, 내용, 작성자, 작성일을 저장하며 첨부파일과 연결된다.
│  │  │
│  │  ├─ post_file.py
│  │  │  # 게시글 첨부파일 테이블 모델
│  │  │  # 원본 파일명, 저장 파일명, 저장 경로, 파일 크기를 저장한다.
│  │  │
│  │  ├─ detection.py
│  │  │  # 객체 탐지 결과 테이블 모델
│  │  │  # 라벨, 신뢰도, 타겟 일치 여부, 알림 여부,
│  │  │  # 카메라 정보, bbox 좌표, 탐지 시각을 저장한다.
│  │  │
│  │  └─ __init__.py
│  │     # 모델 import 묶음 파일
│  │     # Flask-Migrate가 전체 모델을 인식하기 쉽게 한다.
│  │
│  ├─ repositories/
│  │  ├─ user_repository.py
│  │  │  # 회원 DB 접근 계층
│  │  │  # 회원 저장, 조회, 관리자 승격, 삭제를 담당한다.
│  │  │
│  │  ├─ contact_repository.py
│  │  │  # 문의 DB 접근 계층
│  │  │  # 문의 저장, 전체 조회, 사용자별 문의 조회를 담당한다.
│  │  │
│  │  ├─ post_repository.py
│  │  │  # 게시글 DB 접근 계층
│  │  │  # 게시글 저장, 검색, 정렬, 페이징, 상세 조회, 수정, 삭제를 담당한다.
│  │  │
│  │  ├─ post_file_repository.py
│  │  │  # 첨부파일 DB 접근 계층
│  │  │  # 게시글 첨부파일 저장, 조회, 삭제를 담당한다.
│  │  │
│  │  ├─ detection_repository.py
│  │  │  # 탐지 결과 DB 접근 계층
│  │  │  # 탐지 저장, 전체 조회, 라벨 검색, 카메라별 조회, alert 조회, 상세 조회를 담당한다.
│  │  │
│  │  └─ __init__.py
│  │     # repositories 패키지 초기화 파일
│  │
│  ├─ routes/
│  │  ├─ main_routes.py

│  │  │

│  │  │
│  │  ├─ contact_routes.py
│  │  │  # 문의 작성 및 문의 내역 조회 라우트
│  │  │
│  │  ├─ admin_routes.py
│  │  │  # 관리자 전용 문의 관리, 회원 관리 라우트
│  │  │
│  │  ├─ member_routes.py
│  │  │  # 회원 전용 계정 정보 및 내 문의 내역 라우트
│  │  │
│  │  ├─ post_routes.py

│  │  │
│  │  ├─ detection_routes.py
```
