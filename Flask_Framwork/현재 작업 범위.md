
### 현재까지 작업한 트리 구조
```
minimalapp/
├─ run.py                                  # 프로젝트 실행 진입점 (create_app 호출 후 Flask 서버 실행)
│
├─ app/
│  ├─ __init__.py                          # 앱 팩토리 파일 (Flask 앱 생성, 설정 적용, 확장 초기화, 블루프린트 등록)
│  ├─ config.py                            # 환경설정 관리 (SECRET_KEY, DB 연결, 메일 설정 등)
│  ├─ extensions.py                        # Flask 확장 객체 관리 (SQLAlchemy, Migrate, Mail)
│  │
│  ├─ common/                              # 여러 계층에서 공통으로 사용하는 기능 모음
│  │  ├─ __init__.py                       # common 패키지 초기화 파일
│  │  ├─ auth.py                           # 로그인 여부, 관리자 여부 확인 및 현재 사용자 정보 조회
│  │  ├─ exceptions.py                     # ValidationError 등 커스텀 예외 정의
│  │  ├─ response.py                       # render / redirect / json 응답 공통 처리
│  │  ├─ session_manager.py                # 로그인/로그아웃 및 세션 데이터 저장·조회 전담
│  │  └─ utils.py                          # 문자열 정리 등 보조 유틸 함수 모음
│  │
│  ├─ models/                              # DB 테이블 구조 정의 (ORM 모델 계층)
│  │  ├─ __init__.py                       # 모델 묶음 import용 파일
│  │  ├─ user.py                           # 사용자(User) 테이블 모델 정의
│  │  └─ contact.py                        # 문의(Contact) 테이블 모델 정의
│  │
│  ├─ repositories/                        # DB 접근 전용 계층 (조회/저장/수정/삭제 담당)
│  │  ├─ __init__.py                       # repositories 패키지 초기화 파일
│  │  ├─ user_repository.py                # 사용자 데이터 조회, 저장, 권한 변경, 삭제 처리
│  │  └─ contact_repository.py             # 문의 데이터 조회 및 저장 처리
│  │
│  ├─ routes/                              # HTTP 요청을 받는 라우트 계층
│  │  ├─ __init__.py                       # routes 패키지 초기화 파일
│  │  ├─ admin_routes.py                   # 관리자 기능 라우트 (문의 관리, 회원 관리, 권한 부여, 추방 등)
│  │  ├─ auth_routes.py                    # 로그인, 회원가입, 로그아웃 라우트
│  │  ├─ contact_routes.py                 # 문의 작성 및 완료 처리 라우트
│  │  ├─ main_routes.py                    # 홈 화면 라우트
│  │  └─ member_routes.py                  # 회원 전용 라우트 (내 계정, 내 문의 내역 등)
│  │
│  ├─ services/                            # 실제 비즈니스 로직 처리 계층
│  │  ├─ __init__.py                       # services 패키지 초기화 파일
│  │  ├─ auth_service.py                   # 회원가입/로그인 검증 및 처리 로직
│  │  ├─ contact_service.py                # 문의 입력값 검증 및 저장 로직
│  │  └─ mail_service.py                   # 문의 접수 시 관리자 메일 발송 처리
│  │
│  ├─ static/                              # 정적 파일 모음 (CSS, 이미지 등)
│  │  ├─ css/
│  │  │  ├─ admin.css                      # 관리자 화면 전용 스타일
│  │  │  ├─ auth.css                       # 로그인/회원가입 화면 전용 스타일
│  │  │  ├─ base.css                       # 전체 기본 스타일 초기화
│  │  │  ├─ components.css                 # 버튼, 폼, 드롭다운 등 공통 컴포넌트 스타일
│  │  │  ├─ contact.css                    # 문의 관련 화면 전용 스타일
│  │  │  ├─ home.css                       # 홈 화면 전용 스타일
│  │  │  ├─ layout.css                     # 공통 레이아웃 스타일 (header, container, page-top 등)
│  │  │  └─ member.css                     # 회원 화면 전용 스타일
│  │  │
│  │  └─ images/
│  │     ├─ contact.png                    # 문의 화면 배경/시각 요소 이미지
│  │     └─ main.png                       # 홈 화면 메인 이미지
│  │
│  └─ templates/                           # Jinja2 템플릿 모음
│     ├─ admin/                            # 관리자 화면 템플릿
│     │  ├─ inquiry/
│     │  │  ├─ detail.html                 # 관리자 문의 상세 + 답변 저장 화면
│     │  │  └─ list.html                   # 관리자 전체 문의 목록 화면
│     │  └─ user/
│     │     ├─ detail.html                 # 관리자 회원 상세 화면
│     │     └─ list.html                   # 관리자 전체 회원 목록 화면
│     │
│     ├─ auth/                             # 인증 화면 템플릿
│     │  ├─ login.html                     # 로그인 화면
│     │  └─ signup.html                    # 회원가입 화면
│     │
│     ├─ common/                           # 공통 템플릿 모음
│     │  ├─ header.html                    # 전체 페이지 공통 헤더
│     │  └─ profile/
│     │     ├─ detail.html                 # 내 계정 정보 조회 화면
│     │     └─ edit.html                   # 내 계정 정보 수정 화면
│     │
│     ├─ contact/                          # 문의 관련 템플릿
│     │  ├─ complete.html                  # 문의 접수 완료 화면
│     │  └─ create.html                    # 문의 작성 화면
│     │
│     ├─ home/
│     │  └─ index.html                     # 홈 화면 템플릿
│     │
│     ├─ layouts/                          # 공통 레이아웃 템플릿
│     │  ├─ admin_base.html                # 관리자 화면 공통 레이아웃
│     │  ├─ auth_base.html                 # 로그인/회원가입 전용 레이아웃
│     │  ├─ base.html                      # 전체 공통 기본 레이아웃
│     │  └─ member_base.html               # 회원 화면 공통 레이아웃
│     │
│     └─ member/                           # 회원 전용 템플릿
│        └─ inquiry/
│           ├─ detail.html                 # 회원 본인 문의 상세 화면
│           └─ list.html                   # 회원 본인 문의 목록 화면
│
├─ instance/
│  └─ minimalapp.db                        # SQLite 데이터베이스 파일 (개발용)
│
└─ migrations/                             # Flask-Migrate가 관리하는 DB 변경 이력
   ├─ alembic.ini                          # Alembic 설정 파일
   ├─ env.py                               # 마이그레이션 환경 설정
   ├─ README                               # 마이그레이션 관련 안내 파일
   ├─ script.py.mako                       # 마이그레이션 스크립트 템플릿
   └─ versions/
      └─ eaf1acdfd631_init.py              # 초기 DB 스키마 생성 마이그레이션 파일
```
